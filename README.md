# wset-quiz
WSET Level 2 exam

## Hosting

This repository can be hosted using GitHub Pages.

1. Rename or keep the file as `index.html` in the repository root.
2. In your GitHub repository settings, enable GitHub Pages from the `main` branch and root folder.
3. The app will be available at:

   `https://<your-github-username>.github.io/wset-quiz/`

If you want to keep the original file name, you can still access it at:

   `https://<your-github-username>.github.io/wset-quiz/wset-quiz.html`
